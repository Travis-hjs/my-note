项目工程说明

当前工程项目将替代原本/example目录下的所有示例笔记，/example目录依然保留，后续内容新增将在该项目目录下进行。

工程目录依然保持传统html + js + css多页目录模式。

区别原来/example目录下的代码，js 换为 ts，css 换为 scss，方便复制到其他主流前端项目工程中使用。

页面目录配置为src/pages/下，新建页面目录时，需要修改以下 3 处：

1. vite.config.mts 中的 getBuildConfig 方法，添加对应的页面目录构建工程。

2. package.json 中的构建命令，参照已有的复制修改即可。

3. src/main.ts 中的页面入口导航。

项目构建在 package.json 中预设了指定构建某个页面目录的命令方式，其结构输入依然保持传统的html + js + css。
